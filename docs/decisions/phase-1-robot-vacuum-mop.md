## Notes

Constraints: 80 m² laminate + wood-panel floors, no pets, ≤4,000 DKK ceiling, stretch toward auto-empty + auto-mop-wash dock if discount allows, Matter preferred (preserves ecosystem flexibility — see .github/copilot-instructions.md). Caveats from research: Matter support could not be verified from a fetchable manufacturer page for any candidate; if Matter is a hard requirement re-scope to SwitchBot S10 / Roborock S8 MaxV Ultra (well over budget). Suction Pa numbers verified only for MOVA E30 Ultra (7000 Pa); other Pa figures unverified. Warranty rows are Danish statutory minimum (24 mdr, købeloven §83) unless extended — extensions unverified.
